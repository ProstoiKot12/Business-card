# Документация business-card-bot

Добро пожаловать в документацию проекта **Telegram бот-визитка (business-card-bot)**.

Этот проект представляет собой бота-портфолио, написанного на Python с использованием современных асинхронных библиотек и подходов. Бот предназначен для демонстрации ваших проектов (кейсов), контактов и ссылок на соцсети, а также включает удобную встроенную панель администратора для управления контентом прямо из Telegram.

## Основные возможности

- **Публичная часть:**
  - Просмотр портфолио в виде карусели или списка кейсов.
  - Удобная навигация с помощью Inline-клавиатур.
  - Отображение ссылок на владельца (Telegram, GitHub и т.д.).
- **Админ-панель (доступна только пользователям из `ADMIN_IDS`):**
  - Добавление новых кейсов через машину состояний (FSM).
  - Редактирование отдельных полей существующих кейсов (название, стек, ссылки, медиа и т.д.).
  - Удаление кейсов (с подтверждением).
  - Изменение порядка отображения кейсов.

## Технологический стек

Проект построен на следующих технологиях:

- **Язык**: Python 3.11+
- **Фреймворк**: [Aiogram 3.x](https://docs.aiogram.dev/en/latest/) (асинхронный фреймворк для Telegram Bot API)
- **База данных**: PostgreSQL 16
- **ORM**: [SQLAlchemy 2](https://docs.sqlalchemy.org/en/20/) (в асинхронном режиме с драйвером `asyncpg`)
- **Миграции**: [Alembic](https://alembic.sqlalchemy.org/en/latest/)
- **FSM Storage**: Redis 7
- **Конфигурация**: `pydantic-settings`
- **Логирование**: `loguru`
- **Тестирование**: `pytest`, `pytest-asyncio`
